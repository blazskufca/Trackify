About

Trackify is a simple web app/website to track prices of items on e-commerce stores. It currently supports bigbang.si and amazon.de!

Online vendors REGULARLY employ various unfair pricing tactics (such as increasing price before the holiday season, and then offering an artificial "discount") which are more often than not harmful to YOU, the consumer.
This website aims to combat such tactics by keeping track of daily prices.

All you need to do get started is to paste in a link from one of the supported online vendors.
If the item is ALREADY in the database a daily price graph will be shown to you.
If item is not in the database it will be added to the database and it's price will be tracked daily. Come back in a day, week, etc...To see how the price changed through time.
